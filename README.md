<h2 style="margin-top:40px;">About the Project</h2>
<p style="line-height:1.6; font-size:16px;">
    This chatbot is a small experiment I built while learning how to work with basic web technologies. 
    The idea was to create a clean interface where the user can type messages and get simple responses without 
    depending on any external backend. The project focuses mainly on structure and UI styling rather than complex logic.
</p>

<h2 style="margin-top:30px;">Features</h2>
<ul style="line-height:1.7; font-size:16px;">
    <li>Responsive layout with a minimal chat interface</li>
    <li>Light/Dark theme toggle</li>
    <li>Clean UI with simple message bubbles</li>
    <li>Beginner-friendly JavaScript implementation</li>
</ul>

<h2 style="margin-top:30px;">Tech Used</h2>
